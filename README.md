# 社区分类器使用说明
下载并解压缩 “这就是最后的社区分类器了_3月5号.7z”

## 环境要求
- Python 3.10 或更高版本
- 互联网连接（首次运行需要安装依赖）

## 使用方法
1. 双击运行 `启动社区分类器.cmd`
2. 等待脚本检查环境并安装依赖（首次运行需要）
3. 打开浏览器，访问 `http://127.0.0.1:5000`
4. 在网页界面中勾选患者的症状、舌象和脉象特征
5. 点击"预测社区"按钮查看结果

## 功能说明
- **社区分类**：根据患者的症状、舌象和脉象特征，预测其所属的社区类别（1-5），分别是
- 1 社区一(疫病初起，疫毒袭表，卫气同病)
- 2 社区二(疫病中期，邪犯肺脾，气机阻滞)
- 4 社区四(疫病中期，热毒闭肺，肺气痹阻)
- 5 社区五(疫病极期，邪气内闭，正气外脱)
- 3 社区三(疫病后期，肺脾两虚，余邪未尽)
- **概率分布**：展示患者属于每个社区的概率，帮助医生了解分类的可信度
- **直观界面**：使用 Bootstrap 框架构建的响应式界面，操作简单直观

## 技术原理
1. 使用《知识图谱特征回归模型》预测知识图谱相关特征
2. 使用《拓扑特征回归模型》预测拓扑相关特征
3. 将两种特征组合后输入《社区分类器》
4. 社区分类器预测患者所属的社区类别（1-5）
5. 返回预测结果和各社区的概率分布

## 注意事项
- 首次运行时，脚本会自动安装所需的依赖包（Flask、pandas、numpy、joblib、xgboost等）
- 应用运行时会占用端口5000，请确保该端口未被其他程序占用
- 关闭应用时，在命令窗口中按 Ctrl+C 退出
- 如有问题，请查看命令窗口中的错误信息

## 包含文件
- `app.py`：社区分类器的主应用脚本
- `启动社区分类器.cmd`：启动脚本
- `graph_feature_mapper_results_optimized`：知识图谱特征回归模型
- `拓扑特征映射模型结果_优化版`：拓扑特征回归模型
- `社区阶段划分分类器结果`：社区分类器模型
